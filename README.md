# 🧮 A Unique Calculator

A modern and interactive **A Unique Calculator** developed using **HTML, CSS, and JavaScript**.  
This project performs basic arithmetic calculations with a clean user interface and responsive design.

---

## 🚀 Features

✔️ Perform basic arithmetic operations  
✔️ Interactive number pad  
✔️ Clear (`C`) functionality  
✔️ Instant result calculation  
✔️ Stylish dark-themed UI  
✔️ Simple and beginner-friendly project  

---

## 🛠️ Technologies Used

| Technology | Purpose |
|------------|----------|
| HTML5 | Structure of the application |
| CSS3 | Styling and layout |
| JavaScript | Calculator functionality |

---

## 📂 Project Structure

```bash
📁 A Unique Calculator
│── cal.html      # Main HTML structure
│── cal.css       # Styling file
│── cal.js        # Calculator logic
```

---

### Calculator Interface Includes:
- Numeric buttons (0–9)
- Arithmetic operators (+, -, *, /)
- Clear button
- Result display screen

---

## ⚙️ Functionality

### ➕ Input Handling
Users can click on the calculator buttons to enter numbers and operators.

### 🧹 Clear Operation
The `C` button clears the entire display screen.

### 🟰 Expression Evaluation
The `=` button evaluates the mathematical expression and displays the result.

### ❌ Error Handling
Invalid expressions are handled using JavaScript `try-catch` blocks.
